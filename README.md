# Gmail Agent with LangChain

## Setup
1. Install dependencies:h
pip install langchain-openai langchain langchain-core google-auth google-auth-oauthlib google-auth-httplib2 google-api-python-client python-dateutil pydantic

2.Set up Google OAuth:
Download credentials.json from Google Cloud Console
Place it in the project directory

3.Set environment variable:
set OPENAI_API_KEY=your_openai_api_key_here

4.Run the script
python gmail_agent.py
